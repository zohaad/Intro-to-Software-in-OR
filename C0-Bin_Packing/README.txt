This folder contains instances of the
Bin Packing Problem

An instance consists of n items, each having a positive real size.

The input format of the instances is as follows

instanceName
n
s1
s2
.
.
.
sn

Thus every instance file will consitst of n+2 lines.
Example of an instance called "instance1" with 10 items

instance1
10
.12
.2
.3
.1
.4
.5
.22
.33
.8
.8
.6
.14
.22
